# WiFi Logger - Java Project

## Description
This project scans available WiFi networks and logs their details into a file periodically using Java.
It demonstrates Operating System concepts like Process Management, Threading, Scheduling, and File Systems.

## How to Run
- Ensure Java JDK is installed.
- Compile: `javac src/*.java`
- Run: `java -cp src Main`

## Features
- Background scanning
- Logging to a text file
- Scheduling tasks every 30 seconds
